<h1> 📚Programação Modular </h1>

Este repositório foi criado para armazenar exercícios, testes e exemplos de Programação Modular que estou aprendendo durante a faculdade.

A ideia é usar este espaço como um ambiente de prática, onde vou adicionando códigos conforme avanço nos conteúdos da disciplina, praticando conceitos fundamentais de programação.

<h1> 🎯 Objetivo </h1>

O objetivo deste repositório é:

Praticar lógica de programação

Aplicar conceitos de programação modular

Organizar exercícios feitos na faculdade

Acompanhar minha evolução no desenvolvimento

<h1> 🧠 Conceitos praticados </h1>

Alguns conceitos que aparecem nos exercícios deste repositório:

Criação de funções

Separação de responsabilidades no código

Manipulação de strings

Uso de laços de repetição (for, while)

Uso de condicionais

Reutilização de código

<h1> 🚀 Tecnologias utilizadas </h1>

JavaScript

Node.js (para executar os arquivos)
